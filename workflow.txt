If need truffle init: 
truffle init

Write your Solidity smart contracts in the contracts/ directory. Compile them using Truffle:
truffle compile

Start Ganache CLI to run a local Ethereum blockchain:
ganache-cli --port 8545 --gasLimit 6721975 --accounts 10 --defaultBalanceEther 100

Deploy your smart contracts to the local blockchain (Ganache):
truffle migrate --network development

Use Truffle console to interact with your deployed contracts:
truffle console --network development

Write and run tests for your contracts using Truffle:
truffle test
